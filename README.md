## Overview

AuditionKit is delivered as a hosted build artifact, not as a conventional packaged desktop app. The project is meant to publish HTML in a browser-friendly form, with FileMaker WebDirect compatibility as the primary target for delivery.

This approach fits projects that need a dedicated web access point and want to publish it through GitHub Pages. It keeps the release footprint minimal while still giving users a stable place to reach the current build.
